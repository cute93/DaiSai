## 작업규칙

## 프로젝트 개요

* 안드로이드앱 (MyApp)
* 언어: Kotlin
* 최소 SDK: 26 / 대상 SDK: 35
* 애플리케이션 ID : com.ccteacher.daisai

## 빌드 명령어

./gradlew assembleDebug       # 디버그 APK 빌드
./gradlew assembleRelease     # 릴리스 APK 빌드
./gradlew build               # 전체 빌드 (모든 변형)
./gradlew lint                # 린트 검사 실행
./gradlew test                # 단위 테스트 실행
